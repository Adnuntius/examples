Synchronous Ad Calls

This page shows two methods by which synchronous ad calls can happen.

First method is just this HTML code directly in the page:

<script src="http://delivery.adnuntius.com/i?tzo=-660&auId=000000000007eb8e&examples=script&format=script"></script>

The important part in the script src attribute is format=script.

Second method is the following script directly in the page:


<script>
  adn.request({auId: '000000000007eb8e', kv: {examples: ['script']}, format: 'script'});
</script>

Note: for this second method to work, adn.js must have been loaded before this script is executed, which is best assured when adn.js is loaded synchronously.

Further note: a target element isn't required because the ad content is put it into the page directly where the ad request is made.
